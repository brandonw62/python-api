# python-api

## Setup And Installation

```bash
git clone <repo-url>
```

### Setup Python Virtual Env

```bash
python3 -m venv venv
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```

### Update Packages

```bash
pip install pip -U
pip install setuptools -U
pip install pylint
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
